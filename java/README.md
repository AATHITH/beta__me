# Install JAVA
This page will show you **How to install JAVA in UBUNTU 18.04**
## Run these Commands:
```
sudo add-apt-repository ppa:openjdk-r/ppa
sudo apt-get update
sudo apt-get install openjdk-8-jdk
java -version
```
